# interjection
Bash script for filling in the whole "I'd just like to interject for a moment..." copypasta's sub-system name.

./interjection --help # show usage

usage: interjection <sub-system> [MODE]
<sub-system>: "<What you're referring to as <sub-system>, is in fact, GNU/<sub-system>..."
[MODE]: 0 - "I'd just like to interject for a moment..."
[MODE]: 1 - "No, Richard, it's <sub-system>, not 'GNU/<sub-system>'."
